* 复习题


1. C语言的基本模块是什么？\\
   C语言的基本模块是函数。

2. 什么是语法错误？\\
   语句不符合C语言的语法规范引起的错误。

3. 什么是语义错误？\\
   在功能和逻辑含义的表达上存在错误，引起的功能或者结果出现错误。

4. 修改下面的C语言？\\
   正确如下：
   #+begin_src C
     #include <stdio.h>

     int main(){
       int s;

       s = 56;
       printf("There are %d weeks in a year.\n",s);

       return 0;
     }
   #+end_src
